## Upgrading

1. Go to Account Settings → Billing
2. Click "View Plans"
3. Select your new plan
4. Complete Stripe checkout
5. Changes take effect immediately

## Downgrading

1. Go to Account Settings → Billing
2. Click "Manage Billing"
3. Select a lower plan
4. Changes take effect at the end of the current billing period

## Annual Billing

Save 20% with annual billing. Select "Yearly" on the pricing page.

## Feature Gating

Features are enforced server-side:
- OCR level (basic/standard/advanced/enterprise)
- Vision models (Starter+)
- Deep Research (Starter limited, Pro+ unlimited)
- Agents (0/2/10/unlimited)
- API Access (Starter+)
- Audit Logs (Pro+)
- RBAC (Pro+)
- SSO (Team optional, Enterprise full)
- SLA (community/email/priority/24/7/dedicated)
