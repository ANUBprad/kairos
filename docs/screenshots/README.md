## How to Capture

1. Start the development server: `npm run dev`
2. Use a screen recording tool (OBS, LICEcap, or Kap)
3. Record at 1920x1080 or 2560x1440
4. For GIFs, use [Kap](https://getkap.co/) or [LICEcap](https://www.cockos.com/licecap/)
5. Optimize GIFs with [gifski](https://gifski.com/) or [ezgif.com](https://ezgif.com/optimize)
6. Keep GIFs under 5MB for README performance

## Naming Convention

- Use lowercase with hyphens: `rag-chat.png`
- Use descriptive names matching the feature
- PNG for static screenshots, GIF for animations
